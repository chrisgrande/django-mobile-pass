# Introduction

`django-mobile-pass` is a Django package for mobile wallet passes. It supports Apple Wallet `.pkpass` generation and signing, Google Wallet class/object creation, pass persistence, Apple device registrations, Google save/remove callbacks, and push update flows.

Use it when a Django app needs tickets, coupons, boarding passes, loyalty cards, generic cards, or store cards that users can add to Apple Wallet or Google Wallet.
