# Django Mobile Pass

Generate Apple Wallet and Google Wallet passes from a Django 5.2+ app. The package includes builders, Django models, PassKit web service URLs, Google ECv2 callback handling, signed Apple downloads, wallet update notifications, and storage helpers.
